# get-involved

Statische Single-Page, die mit einem Klick eine vorbereitete E-Mail an
den Bundesfinanzminister im lokalen Mail-Client öffnet. Thema: Forderung
nach einer zentralen Bundesstelle gegen international organisierte
Steuerhinterziehung (Cum-Ex / Cum-Cum).

Kein Tracking, keine Cookies, kein Backend.

## Stack

- Vanilla HTML, CSS, JavaScript
- Hosting: Netlify

## Lokal ausführen

Reine statische Seite, kein Build. Einfach `index.html` im Browser
öffnen oder einen statischen Server starten:

```bash
python3 -m http.server 8000
```

## Deployment

Wird via Netlify aus dem `main`-Branch deployed.

## Lizenz

- **Code**: MIT (siehe [LICENSE](LICENSE)).
- **E-Mail-Text**: frei verwendbar, abänderbar und weiterverbreitbar.
  Keine Namensnennung erforderlich. Wer den Text für eigene Aktionen
  oder andere Empfänger nutzen will, soll das tun.
